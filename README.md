# EngSoft I V0.1

> Projeto da disciplina de Engenharia de Software I

#### Clone/Configure

`$ git clone https://github.com/PaulloClara/engsoft_i.git`\
`$ cd engsoft_i`\
`$ python -m venv .env`\
`$ source .env/bin/activate`\
`$ pip install -r requirements.txt`

#### Ativar/Desativar Env

`$ source .env/bin/activate`\
`$ deactivate`

#### Rodar App

`$ python -B run.py`

#### Rodar Testes

`$ python -B run.py --tests`
